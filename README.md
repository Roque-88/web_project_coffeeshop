# Triple Espresso

Este proyecto es una demostración de nuestras habilidades avanzadas en HTML y CSS, donde exploramos técnicas modernas para estructurar y estilizar contenido web. A lo largo de este trabajo,
aprendimos y aplicamos conceptos importantes como la organización de archivos siguiendo la metodología BEM, la creación de formularios interactivos y el uso de contenido embebido para enriquecer la experiencia del usuario.

Este proyecto está diseñado para:

Mostrar el uso de HTML y CSS avanzados para construir una interfacez web amigable y bien estructuradas.
Implementar formularios accesibles y funcionales.
Incorporar contenido embebido (como videos, mapas o elementos interactivos) de manera efectiva.
Seguir las mejores prácticas en la organización de archivos mediante la metodología BEM (Block Element Modifier), asegurando un código limpio y escalable.
El propósito principal de este proyecto es reforzar conceptos fundamentales mientras se practica la construcción de un diseño bien estructurado y moderno.

Tecnologías Utilizadas:
HTML5: Para la estructura semántica del contenido.
CSS3: Para estilizar y mejorar la apariencia visual del sitio.
Metodología BEM: Para una organización clara y coherente del código CSS.
Contenido embebido: Incorporación de elementos multimedia como videos, mapas, etc.

Funcionalidades:
Formulario interactivo y accesible.
Uso de contenido embebido como videos.
Diseño responsivo con organización modular basada en BEM.
